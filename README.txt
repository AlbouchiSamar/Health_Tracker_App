# Health Tracker App

## Description
**Health Tracker App** est une application simple et intuitive qui permet aux utilisateurs de suivre trois paramètres essentiels pour leur santé :
- **Consommation d'eau** (en millilitres).
- **Durée d'exercice** (en minutes).
- **Niveau de sucre dans le sang** (en mg/dL).

Les données saisies sont enregistrées localement dans le navigateur à l'aide du `localStorage`, permettant aux utilisateurs de consulter, modifier et supprimer leurs entrées.

---

## Fonctionnalités
- **Saisie et suivi des données** : Ajoutez des données pour chaque jour.
- **Stockage local** : Les données sont conservées dans le navigateur via `localStorage`.
- **Modification et suppression** : Editez ou supprimez facilement les entrées existantes.
- **Interface utilisateur interactive** : Une interface claire et accessible.
